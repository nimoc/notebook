## 想法认知

编程语言是语言作者将解决某一类问题所需要采用的编程方式提炼成编程语言，并加上很多限制来确保使用此语言时候尽可能的与作者设想的编程方式靠拢。（语言太过自由在不同的场景下一定是好事）

> 最常见的是面向对象编程和函数式编程，如果硬是要在一个函数式编程的语言中加入面向对象，会举步艰难。
